# CRISP-DM

## 1. Entendimento do Negócio

O projeto busca transformar dados públicos educacionais em inteligência analítica para apoiar o acompanhamento da alfabetização infantil no Brasil.

## 2. Entendimento dos Dados

A primeira fonte analisada será a tabela de UF da Base dos Dados, relacionada ao Indicador Criança Alfabetizada.

## 3. Preparação dos Dados

A preparação será organizada nas camadas Bronze, Silver e Gold.

## 4. Modelagem

A camada Gold poderá apoiar modelos preditivos e análises de desigualdade educacional.

## 5. Avaliação

Serão avaliadas completude, duplicidade, consistência e aderência às perguntas de negócio.

## 6. Implantação

A solução será documentada no README e poderá ser publicada futuramente na AWS.
